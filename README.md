## XSHARE

A file sharing application that shares any kind of file through your email or converts it to a link.

### Technology/Stack

* React
* Node/Express
* Mongodb

<hr>

### Features
* Upload file
*  Send as email
*  Copy as a link
*  Share on social platforms
